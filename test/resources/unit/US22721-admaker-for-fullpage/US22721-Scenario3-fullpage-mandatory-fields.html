  • Click the 'Copy Reviewed Ad Code' Button.
  • A dialog box with your Ad Code should appear.
  • Press CTRL C and the 'Enter' key simultaneously.
  • Open DFP in Internet Explorer.
  • Go to your Ad Order or create a new one.
  • Select or create your ad.
  • Select or create a new creative of the 'Custom' type
  • Paste the ad code in the 'Creative Code' field.
